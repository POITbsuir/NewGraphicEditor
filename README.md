# NewGraphicEditor 🎨

**NewGraphicEditor** — это учебный проект, созданный для практического закрепления принципов объектно-ориентированного программирования (ООП).

Приложение представляет собой простой графический редактор (дизайнер), который позволяет пользователям создавать и отображать различные 2D и 3D фигуры. Архитектура проекта спроектирована с учетом расширяемости: добавление новых пользовательских фигур не требует изменения основного кода программы.

## 🎯 Цель проекта

Основная цель — не создать профессиональный графический пакет, а на практике реализовать и понять ключевые парадигмы ООП:

*   **Наследование:** Создание иерархии классов (абстрактные базовые классы -> конкретные фигуры).
*   **Инкапсуляция:** Сокрытие внутренних расчетов и данных внутри классов фигур.
*   **Полиморфизм:** Использование единого интерфейса (например, метода `draw()`) для работы с разными типами фигур, будь то круг или трехмерный куб.
*   **Абстракция:** Выделение общих свойств и поведения всех фигур в базовые классы.

## ✨ Возможности

*   Отрисовка базовых **2D-фигур**: круг, квадрат, треугольник и т.д.
*   Отрисовка базовых **3D-фигур**: куб, сфера, пирамида и т.д. (либо их проекции на плоскость).
*   **Расширяемость:** Легко добавлять собственные фигуры, унаследовав их от базового класса `Shape` и реализовав необходимые методы.
